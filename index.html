适用仪器:Agilent 8890 GC + 5977B MSD + MassHunter GC/MS Acquisition Software 编写日期:2026-04-23 用途:从开机到出数据的完整操作流程,按顺序循序查阅

1. 仪器总览与工作原理

1.1 系统数据流

He 气源 → GC 进样口 → 色谱柱(柱箱升温) → 传输线(Aux 2) → MSD 离子源 → 四级杆 → 检测器
  ↑           ↑              ↑                 ↑              ↑           ↑          ↑
0.5 MPa   温度/分流比     升温程序           ~250°C         EI 电离    m/z 过滤    EMV 放大

分工: - GC 负责分离(不同化合物在不同时间出柱) - MSD 负责鉴定(同一时间点的物质,用 m/z 区分)

1.2 三种采集模式的选择

模式 何时用 灵敏度 信息量
Scan(全扫描) 未知物定性、谱库匹配 低 全谱
SIM(选择离子监测) 已知目标物痕量定量 最高(10-100× Scan) 仅目标离子
SIM/Scan 定量+定性兼顾 中等 兼顾

SIM 灵敏度为什么高? Scan 每个 m/z 只用 1/5000 时间;SIM 用 100% 时间只盯一个 m/z。

2. 开机流程

2.1 开机顺序(顺序错误会损坏真空系统)

① 打开 He 钢瓶 → 减压阀输出压力 0.5 MPa
② 打开 GC 电源(柱子先获得载气保护)
③ 用手推住 MSD 侧板凸起方块 → 同时打开 MSD 电源
   └─ 侧板没吸紧会漏真空,推住约 10 秒感觉吸牢再松手
④ 启动 MassHunter Acquisition 软件

原理: - 先通 He 防止柱子在无保护下通电 - MSD 开机瞬间真空泵抽气,侧板需靠负压吸紧

2.2 真空建立判断标准

参数 正常值 位置
Turbo Speed 100% Instrument Control 底部
Hi Vac < 2 × 10⁻⁵ Torr Instrument Control 底部
启动时间 5977B 约 10-15 min;5977A 约 20 min —

真空没到标准前不能开灯丝/调谐,软件有 lockout 保护。

3. 调谐(Tune)

3.1 调谐做什么

用 PFTBA(全氟三丁胺)标准物校准 MS 的质量轴和响应轴。

PFTBA 的三个特征离子:m/z = 69 / 219 / 502(必记,横跨低中高质量段)

3.2 调谐文件选择

文件 适用离子源
Atune.u EI 源(Inert 或不锈钢),默认首选
Etune.u Extraction 源(5977 专用,带拉出透镜)
HES_Atune.u HES 高灵敏度源
Stune.u 偏重质量轴精度和分辨率
BFB.u / DFTPP.u EPA 524/525 法规方法
Pcich4.u / Ncich4.u CI 模式(甲烷)

选择原则:文件类型必须匹配实际装的离子源。

3.3 调谐操作路径

Instrument Control
  → View 菜单 → Tune and Vacuum Control
  → Tune 菜单 → 选对应调谐类型(如 Extraction Source Tune)
  → 自动调谐,约 5-10 min
  → Tune 菜单 → Tune Evaluation → 查看评价报告

3.4 调谐报告判读(考核高频)

① 三个目标离子

Target m/z   Actual m/z   判断
69.00        69.00 ± 0.1  ← 基峰参考
219.00       219.00 ± 0.1 ← 中质量段
502.00       502.00 ± 0.1 ← 高质量段

② 丰度比(合格硬指标)

比值 合格范围
219/69 32-54%
502/69 > 2.4%
同位素 70/69 ~1.1%
同位素 220/219 ~4.5%
同位素 503/502 ~10%

③ Air/Water Check(检漏关键)

比值 含义
H₂O/69 新抽真空会很高,会慢降
N₂/69 < 20% 超过 = 漏气
O₂/69 < 10% 超过 = 漏气

口诀:"28 和 32 高 = 漏气"(m/z=28 是 N₂,m/z=32 是 O₂)

④ EM Volts(电子倍增器电压)

  • 健康值:< 2000 V
  • 接近 3000 V:该换 EM 了
  • 值随仪器老化上升(补偿机制)

3.5 什么时候重新调谐

  • 清洗或更换离子源后
  • 更换色谱柱后
  • 仪器长时间停机后重启
  • 数据响应异常降低或质量数漂移

4. 方法编辑(Method)

入口:Method → Edit Entire Method → 勾选"方法信息"和"仪器/采集"

4.1 ALS 自动进样器

参数 典型值 原理/调节逻辑
进样量 1 μL 太大会溶剂膨胀倒灌;太小定量不准
溶剂 A 清洗 3 次(进样前+后) A 用样品溶剂(如己烷)
溶剂 B 清洗 3 次 B 用强极性洗剂(甲醇)
样品抽吸次数 6 次 排气泡,保证进样量准确
柱塞速度 快速(默认) 慢速用于挥发性极高样品

蒸汽体积计算器:检查汽化后是否超过 Liner 体积(建议 ≤30%)。

4.2 进样口(Inlet)

参数 典型值 说明
加热器温度 250-280 °C 高于最高沸点化合物 30-50 °C
进样模式 见下表 决定灵敏度与过载平衡
隔垫吹扫流量 3 mL/min 防止隔垫降解物污染柱
载气节省 开启,3 min 后启用 20 mL/min 省 He,不影响分离

进样模式选择:

模式 场景
Split 样品浓度高(分流比 50:1 常见)
Splitless 痕量分析,样品全部进柱
Pulsed Splitless 痕量 + 高沸物

Splitless 关键附加参数: - 分流出口吹扫流量:60 mL/min - 吹扫时间:1 min(之后打开分流阀吹走残留溶剂蒸汽)

4.3 色谱柱(Column)

参数 设定 原理
流量控制模式 恒流量(Constant Flow) GC-MS 必选,保证 MS 真空稳定
流量 1.0-1.5 mL/min He > 2 会超 MS 真空承受
出口压力 真空(Vacuum) 柱出口接 MSD 真空

4.4 柱箱升温程序(分离核心)

标准结构:

初始温度 40 °C 保 1 min
  ↓ 斜率 10 °C/min
150 °C 保 5 min
  ↓ 斜率 15 °C/min
260 °C 保 2 min
参数 调节逻辑
初温 低于溶剂沸点 20 °C(如己烷 BP=69 → 初温 40)
升温斜率 10-20 °C/min 常规;快 = 出峰快但分辨率降
终温 不超过柱子最高使用温度(如 HP-5ms: 325 °C)
终温保持 ≥ 2 min 清柱,防残留污染

4.5 辅助加热器 Aux 2(MSD 传输线)

  • 典型值:250-280 °C
  • 原则:≥ 柱温最高点,≤ 柱子最高使用温度
  • 作用:防止化合物在传输线冷凝

4.6 MS 采集参数

① 调谐文件:从下拉选,匹配离子源类型。

② 溶剂延迟(Solvent Delay) — 最关键

  • 典型值:3.0-3.5 min
  • 含义:前 3 min 不点灯丝,等溶剂峰过
  • 设置方法:溶剂延迟 = 溶剂保留时间 − 0.3 min
  • 设短了 → 烧灯丝;设长了 → 错过早出目标峰

③ EMV 模式

模式 用法
Gain Factor 默认首选,相对调谐电压的倍数(1.0 = 用调谐值)
Absolute Voltage 直接指定电压,不推荐
Relative Voltage 调谐电压±增量,临时提升灵敏度

④ 离子源/四级杆温度

  • 一般不改,用调谐文件设定值
  • EI 源:230 °C / 150 °C
  • HES 源:300 °C / 200 °C

⑤ Scan 模式参数

参数 典型值 说明
扫描起点-终点 m/z 35-550 避开空气峰(28,32),覆盖常见有机物
阈值 0 或 150 低保留更多信息,高去噪
采样数(Sampling) 2 平衡速度与灵敏度
扫描次数 每个色谱峰 ≥ 10 个数据点 保证峰形完整

⑥ SIM 模式参数

参数 典型值 说明
离子列表 每化合物 2-4 个 1 个定量 + 1-3 个定性
驻留时间 50-100 ms 越长信号越强,但总循环慢
分组(Group) 按保留时间分段 不同时段监测不同离子

⑦ 时间段(Time Segments)

一个方法内可设多段,每段不同扫描范围或 SIM 离子组,提升效率。

4.7 方法保存

File → Save As → 命名规则:化合物_模式_日期.m
示例:BTEX_SIM_20260423.m

.m 是文件夹,内含所有参数 XML。

5. 跑样(单针 + 序列)

5.1 跑样前的就绪检查

三个状态必须全绿(Instrument Control 窗口):

Run Status:        Idle
Instrument Status: Ready

底部实时参数全部到位: - Oven Temperature = 初温 - Inlet Temperature = 设定温度 - Column Flow = 设定流量 - MS Source / Quad 温度 = 调谐设定 - Hi Vac < 2e-5 Torr - Turbo Speed = 100%

5.2 单针运行(Run Method)

1. Method → Load Method → 选方法
2. 等 Ready
3. 点绿色箭头(Run Method)
4. 填开始运行对话框:
   ├─ Operator(操作员)
   ├─ Data Path(建议按项目建子目录)
   ├─ Data File(格式:日期_样品ID_序号.D)
   └─ Vial Number(核对实际瓶位!)
5. 确定并运行方法

数据文件 .D:本质是文件夹,内含 data.ms + 方法副本 + 采集日志。拷贝时拷整个文件夹。

5.3 序列运行(Sequence)

操作:

1. Sequence → Load Sequence 或 New Sequence
2. 逐行填样品信息
3. Save Sequence As(命名:项目_批次.sequence.xml)
4. 点"运行序列"图标
5. 确认运行选项

序列表关键列:

列 填法
样品瓶位 对应实际样品盘位置
样品名 Sample_1, STD_5ppm, BlankH2O
方法路径 选 .m 文件
数据文件 自动递增或手动
类型 Sample / Calibration / Blank / QC
级别 校正曲线浓度级别
稀释因子 浓度计算用

经典序列编排套路:

Vial 1:  Solvent Blank        (确认基线干净)
Vial 2-6: Std 低→高           (标准曲线,浓度递增)
Vial 7:  Solvent Blank        (验证无残留)
Vial 8:  QC                    (验证校正)
Vial 9-11: Samples
Vial 12: QC                    (每 10 针插一个,监测漂移)
Vial 13-: Samples
Vial last: Solvent Blank      (清柱收尾)

要点: - 浓度从低到高(避免残留污染) - 每隔一段插 QC - 标准品和样品之间插 Blank - 序列首尾都放 Blank

运行序列选项: - 覆盖现有数据文件 → 不勾(默认安全) - 后序列宏 → CI 模式可加 CIGASOFF 自动关反应气

5.4 运行中的监控

看什么 异常 → 可能原因
基线高度 持续升高 → 柱流失/污染
溶剂峰大小 突然变小 → 进样器问题或空瓶
保留时间 逐针漂移 → 流量/柱温不稳
目标峰面积 逐针下降 → 离子源污染/灯丝老化

5.5 中途操作

  • Pause Sequence:当前针完成后暂停
  • Stop Sequence:当前针完成后停止
  • Abort Run(STOP 按钮):立即中止,数据部分保存

6. 关机流程

严格反向顺序(绝不能先关气):

① Tune Vacuum Control → 快速放空(Vent)
   └─ 系统自动调用 vent method:降 GC 温度、关灯丝、降涡轮转速
② 等离子源温度降到 < 100 °C
③ 关 MSD 电源
④ 关 GC 电源
⑤ 最后关 He 钢瓶

为什么不能先关气? 没有 He 正压,空气会从进样口倒灌,污染柱子和离子源。

使用 H₂ 载气时另需参考《氢气安全手册》。

7. 常见异常与排查

7.1 真空起不来

现象 原因 处理
Turbo Speed 长时间 < 100% 侧板没吸紧 重启,开机时再次推紧侧板
Hi Vac 数值持续高 漏气 检查进样口隔垫、分流出口、柱连接
涡轮泵声音异常 机械故障 报修

7.2 调谐失败

症状 可能原因
69/219/502 丰度太低 PFTBA 快用完 / 离子源脏
N₂/O₂ 比超标 系统漏气(优先排查进样口隔垫)
EM Volts > 2500 电子倍增器老化,考虑更换
质量轴偏移 > 0.2 amu 需重新调谐或联系工程师

7.3 跑样错误

错误提示 处理
"进样器无法吸样" 检查样品瓶液量、瓶盖、针是否堵塞
"MS 未就绪" 检查真空、温度、灯丝
"GC 未就绪" 检查载气、柱温、进样口温度
数据无峰 检查溶剂延迟设置、进样是否成功、调谐是否正常

7.4 数据质量异常

现象 排查
峰形拖尾 进样口温度低、Liner 脏、柱子老化
峰面积不重复 进样针问题、气泡、样品挥发
基线漂移 柱流失(到柱温上限)、载气污染
灵敏度突降 离子源脏、灯丝老化、传输线污染

8. 关键数值与原则速查

8.1 必记数值

项目 值
PFTBA 三个特征离子 69, 219, 502
调谐 502/69 比合格下限 > 2.4%
Air/Water 检漏 N₂/69 < 20%
Air/Water 检漏 O₂/69 < 10%
健康 EM Volts < 2000 V
He 减压阀输出压力 0.5 MPa
Hi Vac 正常值 < 2 × 10⁻⁵ Torr
柱流量(GC-MS) 1.0-1.5 mL/min
溶剂延迟 3.0-3.5 min
SIM 驻留时间 50-100 ms

8.2 核心原则

  1. 开机:先气后电(He → GC → MS);关机:先断气断前放空、气最后关
  2. GC-MS 必须恒流量模式,不用恒压力
  3. 色谱柱出口必须选 Vacuum(接 MSD)
  4. 溶剂延迟不能设短,否则烧灯丝
  5. 传输线温度 ≥ 柱温最高点
  6. 调谐文件类型要匹配实际离子源
  7. 序列:浓度低→高,首尾 Blank,中间插 QC
  8. 数据 .D 是文件夹,整体拷贝

8.3 快捷决策

  • 未知物 → Scan 模式,谱库检索
  • 痕量定量 → SIM 模式,Splitless 进样
  • 法规方法(EPA)→ 用 BFB/DFTPP 调谐
  • 漏气判断 → 看调谐报告的 N₂ 和 O₂ 占 69 的比
